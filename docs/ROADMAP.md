# Roadmap

These milestones are ordered to make the project increasingly attractive in interviews.

## Near Term

- Add unit tests for `UrlTools`, bookmark serialization, and history compaction.
- Add bookmark editing and deletion from the UI.
- Expand the page information dialog with certificate and permission details.
- Extend settings for download directory, JavaScript preferences, and cache controls.

## Portfolio Polish

- Add a screenshot gallery to the README after building locally.
- Package Windows builds with CPack or the Qt Installer Framework.
- Add a short demo video showing tabs, bookmarks, private mode, downloads, and Developer Tools.
- Add GitHub release workflow artifacts for Windows and Linux.

## Advanced Ideas

- Replace the starter request blocker with filter-list import support such as EasyList syntax.
- Add permission prompts for camera, microphone, notifications, and geolocation.
- Add profile switching for work/personal browsing contexts.
- Add command palette navigation.
- Add crash-safe session persistence.
- Add WebAuthn/passkey and password-manager integration research notes.
